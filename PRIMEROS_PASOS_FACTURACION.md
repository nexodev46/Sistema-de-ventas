# 🚀 PRIMEROS PASOS - FACTURACIÓN PDF

## PASO 1️⃣: CONFIGURAR TUS DATOS (3 minutos)

Abre tu navegador y ve a:  
**Configuración → Facturación**

```
1. Llena DATOS DE FACTURACIÓN (izquierda):

   Razón Social: [Tu nombre de empresa]
   RUC: [Tu número RUC - ej: 12345678901]
   Dirección: [Tu dirección completa]
   Teléfono: [Tu teléfono - ej: +51999999999]
   Email: [Tu email - ej: info@empresa.com]

2. Llena NUMERACIÓN:

   Serie: F001  (por defecto, puedes cambiar)
   Número Inicial: 1  (donde empieza tu numeración)
   
   💡 Si ya facturaste hasta F001-00045, 
      pon "Siguiente Número: 46"

3. Llena IMPUESTOS:

   Nombre del impuesto: IGV  (en Perú, normalmente "IGV")
   Porcentaje (%): 18  (en Perú es 18%)
   ☑ Incluir impuesto en precios  (checkea esto)

4. Llena MONEDA Y FORMATO:

   Moneda: [PEN ▼]  (Perú = PEN, USA = USD, Europa = EUR)
   Decimales: 2

5. EXTRAS (personalizacion):

   Pie de página: ¡Gracias por su compra!
   (Puedes cambiar por algo como "Garantía 1 año")
   
   Términos y condiciones: [Opcional]

6. CLICKEA: [Guardar Cambios] (botón verde abajo)
```

**✅ LISTO**: Tus datos se guardaron. Ahora verás preview en la derecha.

---

## PASO 2️⃣: PROBAR CON EJEMPLO (2 minutos)

Aún en **Configuración → Facturación**

```
EN LA DERECHA ("Vista Previa"), verás 2 botones:

1️⃣ Clickea [Imprimir Prueba]
   → Se abre una vista previa
   → Verás cómo se verá tu factura impresa
   → Cierra la ventana

2️⃣ Clickea [PDF de Prueba]  
   → Se descarga: Factura_F001-00000001.pdf
   → Abre el archivo con tu lector de PDF
   → Verifica que se vea bien
   → Si falta algo, vuelve al Paso 1 y ajusta
```

**✅ LISTO**: Ya viste cómo se verá tu factura en PDF.

---

## PASO 3️⃣: DESCARGAR FACTURA REAL (1 minuto)

Ahora que configuraste todo, vamos a descargar una factura de verdad.

```
1. Ve a: Ventas → Listado de Ventas

2. Busca una venta que hayas hecho (o haz una nueva)

3. En la fila de esa venta, verás 4 botones en "Acciones":
   
   👁️ Ver detalles
   📄 ← ESTE ES EL PDF (rojo/verde)
   🖨️ Imprimir
   🗑️ Eliminar

4. CLICKEA el botón 📄 (el segundo botón)

5. Se descargará automáticamente:
   Factura_F001-00000001.pdf
   
   (El número varía según tu serie y número de factura)

6. Abre el PDF y verifica:
   ✓ Tu nombre de empresa
   ✓ RUC y dirección
   ✓ Número de factura (F001-00000001)
   ✓ Fecha de hoy
   ✓ Nombre del cliente
   ✓ Productos y cantidades
   ✓ Subtotal
   ✓ IGV (18%)
   ✓ TOTAL
   ✓ Tu mensaje personalizado
```

**✅ LISTO**: ¡Tienes tu primera factura en PDF descargada! 🎉

---

## PASO 4️⃣: IMPRIMIR (Opcional, 1 minuto)

Si quieres imprimir en lugar de solo descargar:

```
1. Ve a: Ventas → Listado de Ventas

2. Busca la venta y clickea en detalle (👁️)
   O directamente clickea en el icono de imprimir (🖨️)

3. Se abre un diálogo de impresión

4. Selecciona tu impresora y clickea [Imprimir]

5. ¡Listo! Tu factura se imprime
```

**✅ LISTO**: Factura impresa lista para entregar al cliente.

## 🆘 AYUDA RÁPIDA

### No veo el botón PDF
```
1. Recarga la página (Ctrl + F5)
2. Ve a Ventas → Listado de Ventas
3. Verifica que la venta esté en estado "COMPLETADA" (verde)
4. El botón 📄 debe estar en la columna "Acciones"
```

### PDF se descarga pero está en blanco
```
1. Ve a Configuración → Facturación
2. Completa TODOS los campos (empresa, RUC, etc)
3. Clickea "Guardar Cambios"
4. Intenta descargar de nuevo
```

### El PDF no se abre
```
1. Revisa tu carpeta de descargas
2. El archivo debe llamarse: Factura_F001-XXXXXXXX.pdf
3. Intenta abrirlo con otro programa (Adobe Reader, navegador, etc)
4. Si sigue sin funcionar, intenta descargar "PDF de Prueba" primero
```

### Número de factura está mal
```
1. Va a Configuración → Facturación
2. Mira "Siguiente Número: F001-00000001"
3. Si necesitas que sea F001-00046, cambia a: 46
4. Clickea "Guardar Cambios"
5. Próximas facturas empezarán desde F001-00046
```

## 💾 PRÓXIMAS VECES

Después de configurar UNA SOLA VEZ, simplemente:

```
1. Haces venta en Punto de Venta
2. Vas a Ventas → Listado
3. Clickeas 📄 PDF
4. ¡Descargada! ✅
```

**No necesitas volver a configurar nada** (a menos que cambies empresa, IGV, etc).

---

## 🎓 IMPORTANTE

### ⚠️ Los PDFs se generan CADA VEZ

- Si tienes venta antigua y cambias configuración
- La próxima vez que descargues ese PDF
- Se generará CON LA CONFIGURACIÓN NUEVA
- Es decir: todos los PDFs siempre reflejan tu config actual

### ⚠️ Los números NO se incrementan automáticamente

- Si descargas factura F001-00000001
- No se cambia automáticamente a F001-00000002
- TENDRÁS QUE CAMBIAR MANUALMENTE si quieres incremento
- En Configuración → Facturación → "Siguiente Número: 2"

### ⚠️ Base de datos SIN CAMBIOS

- Los PDFs NO se guardan en la BD
- Se generan bajo demanda
- Solo cuando clickeas botón PDF
- Esto significa: sin costo de almacenamiento extra ✅
